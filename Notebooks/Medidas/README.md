# Projeto Didático: Medidas de Posição e Dispersão em Python

## Introdução

As medidas de posição e de dispersão são ferramentas essenciais na análise de dados. Elas nos ajudam a entender onde os dados estão localizados e o quão espalhados estão, respectivamente. Imagine que você está organizando uma festa e precisa saber onde a maioria dos seus amigos mora e o quão longe alguns deles podem vir. As medidas de posição e dispersão te ajudam nessa tarefa.


### Medidas de posição:

- **Média**: É a soma de todos os valores dividida pelo número total de observações. Em outras palavras, é o valor "médio" dos dados.
- **Moda**: É o valor que aparece com mais frequência em um conjunto de dados. É útil para identificar o valor mais comum.
- **Mediana**: É o valor que está no meio de um conjunto de dados quando eles são ordenados. Se houver um número par de observações, a mediana é a média dos dois valores do meio.

### Medidas de variabilidade:

- **Mínimo**: É o menor valor presente no conjunto de dados.
- **Máximo**: É o maior valor presente no conjunto de dados.
- **Amplitude**: É a diferença entre o valor máximo e o valor mínimo. Indica a variação total dos dados.
- **Quartis**: Os quartis dividem um conjunto de dados ordenados em quatro partes iguais, cada uma representando 25% dos dados.
- **Percentis**: Indica a porcentagem de dados que está abaixo de um determinado valor.
- **Variância**: Mede a dispersão dos dados em relação à média. É calculada como a média dos quadrados das diferenças entre cada valor e a média.
- **Desvio Padrão**: É a raiz quadrada da variância. Indica o quanto os valores do conjunto estão desviando da média.
